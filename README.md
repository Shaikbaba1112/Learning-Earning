# Learning-Earning

A JavaScript practice application built with React and Vite, designed to help developers learn and practice JavaScript concepts through interactive code examples and challenges.

## Overview

This project provides a component-based architecture for organizing and presenting JavaScript practice questions and code examples. It features a responsive UI with navigation, hero section, and organized code cards for different topics.

## Features

- **Interactive Components**: Organized practice questions with syntax highlighting
- **Modern Stack**: Built with React 18 and Vite for fast development and build times
- **Responsive Design**: Mobile-friendly UI with CSS Grid and Flexbox
- **Component-Based Architecture**: Modular, reusable components for easy maintenance

## Tech Stack

- **React** 18.2.0 - UI library
- **Vite** 4.3.0 - Build tool and dev server
- **CSS3** - Styling with modern layouts

## Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/Learning-Earning.git
cd Learning-Earning
```

2. Install dependencies:
```bash
npm install
```

## Development

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Build

Create a production build:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Project Structure

```
src/

│   ├── CodeCard/      # Display individual code examples
│   ├── Hero/          # Landing section
│   ├── Navigation/    # Main navigation
│   └── parts/         # Practice question parts
├── App.jsx            # Main application component
├── index.jsx          # Application entry point
└── styles/            # Global styles
```
